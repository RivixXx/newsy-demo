# Техническое задание

## Проект

Разработка Telegram-бота MVP для платформы NEWSY

## Цель проекта

Создать Telegram-бота, обеспечивающего базовое взаимодействие пользователей с платформой NEWSY через Telegram без необходимости постоянного посещения веб-сайта.

Основная задача MVP — проверить востребованность Telegram-интерфейса, повысить вовлеченность пользователей и обеспечить удобный доступ к челленджам и системе достижений.

---

# Технологический стек

## Backend

* Node.js 22 LTS
* TypeScript
* Telegram Bot API
* Telegraf
* PostgreSQL
* Prisma ORM
* REST API для интеграции с NEWSY

# MVP-функционал

## 1. Авторизация пользователя

### Команда

```text
/start
```

### Логика

При первом запуске:

* создается пользователь;
* сохраняется Telegram ID;
* сохраняется Username;
* сохраняется имя пользователя;
* создается профиль NEWSY.

---

## 2. Профиль пользователя

### Команда

```text
/profile
```

Отображает:

* имя пользователя;
* количество баллов;
* место в рейтинге;
* количество выполненных заданий;
* количество достижений.

---

## 3. Список челленджей

### Команда

```text
/challenges
```

Отображает:

* активные челленджи;
* сроки проведения;
* текущий прогресс;
* количество участников.

---

## 4. Карточка челленджа

После выбора челленджа отображается:

* название;
* описание;
* награда;
* сроки проведения;
* кнопка "Принять участие".

---

## 5. Участие в челлендже

После нажатия:

```text
Принять участие
```

Пользователь добавляется в список участников.

---

## 6. Выполнение задания

Пользователь нажимает:

```text
Выполнить задание
```

В MVP доступны:

* отметка выполнения;
* загрузка фотографии.

---

## 7. Достижения

### Команда

```text
/achievements
```

Показывает:

* полученные достижения;
* прогресс по следующим достижениям.

---

## 8. Рейтинг

### Команда

```text
/rating
```

Отображает ТОП-10 участников.

---

## 9. Уведомления

Система должна поддерживать отправку:

* новых челленджей;
* напоминаний;
* достижений;
* изменения рейтинга.

---

# Административный функционал

Доступен только администраторам.

## Команды

```text
/admin
```

Функции:

* создание челленджа;
* завершение челленджа;
* просмотр статистики;
* массовая рассылка сообщений.
