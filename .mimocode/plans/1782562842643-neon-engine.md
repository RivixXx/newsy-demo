# SEO-оптимизация для NEWSY (chillenge-russia.ru)

## О проекте
NEWSY — интерактивная цифровая платформа челленджей для маркетплейсов. Платформенное решение для бизнеса, блогеров, экспертов и граждан. Категории: Спорт, Обучение, Квесты, Искусство, Технологии. УТП: верифицированные достижения, награды, пошаговые задания, геймификация.

## Текущее состояние
- Только корневой layout имеет статичные метаданные (title='NEWSY', общий description)
- Ни одна страница не экспортирует `generateMetadata` или `metadata`
- Нет `sitemap.ts`, `robots.ts`, JSON-LD, OG-изображений
- Главная — `'use client'` с мок-данными (поисковики не видят контент)
- Страница челленджа — серверный компонент, но данные тоже замоканы

## План (13 файлов, 7 фаз)

### Фаза 1: Извлечь мок-данные в общий модуль
**Новый файл: `src/shared/data/challenges.ts`**
- Вынести `MOCK_CHALLENGES` и `CatalogChallenge` интерфейс из `src/app/(public)/page.tsx`
- Экспортировать `MOCK_CHALLENGES`, `CatalogChallenge`, `getChallengeById(id)`
- Обновить импорт в `src/app/(public)/page.tsx`

### Фаза 2: Улучшить метаданные корневого layout
**Файл: `src/app/layout.tsx`** (edit)
- `metadataBase: new URL('https://chillenge-russia.ru')`
- `title: { default: 'NEWSY — Платформа челленджей для маркетплейсов', template: '%s | NEWSY' }`
- `description: 'Интерактивная платформа челленджей NEWSY. Спорт, обучение, квесты, искусство и технологии. Верифицированные достижения и награды от лучших брендов России.'`
- `keywords: ['челленджи', 'челлендж', 'достижения', 'награды', 'бренды', 'мероприятия', 'квесты', 'спорт', 'обучение', 'искусство', 'технологии', 'активности', 'геймификация']`
- `openGraph: { type: 'website', locale: 'ru_RU', siteName: 'NEWSY', url: 'https://chillenge-russia.ru' }`
- `twitter: { card: 'summary_large_image' }`

### Фаза 3: Метаданные для страниц

**Новый файл: `src/app/(public)/layout.tsx`** (серверный компонент)
- `title: 'Каталог челленджей — Спорт, Обучение, Квесты, Искусство, Технологии'`
- `description: 'Найди челлендж для себя на NEWSY. Верифицированные достижения и награды от лучших брендов. Превращаем любовь к брендам в достижения.'`
- `openGraph` с изображением `/og-home.png`
- JSON-LD: `WebSite` + `Organization` schemas

**Файл: `src/app/(public)/challenges/[id]/page.tsx`** (edit)
- `generateMetadata()` — ищет челлендж по ID в моках
- `title: ${challenge.title} | NEWSY`
- `description`: обрезанный description челленджа (до 160 символов)
- `openGraph`: imageUrl из челленджа
- JSON-LD: `Event` schema в render()

**Файл: `src/app/(public)/search/page.tsx`** (edit)
- `metadata`: title='Поиск челленджей', `robots: { index: false }`

**Файлы: login/register** (edit)
- `robots: { index: false, follow: false }`

### Фаза 4: Компонент JSON-LD
**Новый файл: `src/shared/components/seo-jsonld.tsx`**
- Серверный компонент `<JsonLd data={...} />`
- `dangerouslySetInnerHTML` с `application/ld+json`

### Фаза 5: Sitemap + Robots
**Новый файл: `src/app/sitemap.ts`**
- Статичные: `/` (priority 1), `/search` (priority 0.5)
- Динамические: `/challenges/[id]` (priority 0.8)

**Новый файл: `src/app/robots.ts`**
- `allow: /`, `disallow: ['/admin', '/dashboard', '/api']`
- `sitemap: 'https://chillenge-russia.ru/sitemap.xml'`

### Фаза 6: Security Headers
**Файл: `next.config.js`** (edit)
- `headers()` с X-Frame-Options, X-Content-Type-Options, Referrer-Policy, Permissions-Policy, CSP

### Фаза 7: OG-изображение
**Новый файл: `public/og-home.png`** (1200x630)
- Статичное OG-изображение для главной
